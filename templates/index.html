How does this work?

Writing recipes can be really dull, so this tool was made to streamline the process, so you can focus on making and sharing good food.
To get the best results, start with the name of the recipe, then include ingredients (and optionally how much of each) you're using to make the dish, and some rough steps (don't worry, the final result is editable!)
Then, hit generate, and you should be met with a pro-looking recipe that you can save and share!
